# Wędkarski Dziennik (PWA)

PWA do archiwizowania wypraw wędkarskich (wpisów dziennika) oraz złowionych ryb przypisanych do konkretnej wyprawy. Aplikacja będzie działać offline, wykorzystywać natywne funkcje urządzenia (geolokalizacja, kamera).

## Cel projektu
Stworzyć prostą, działającą PWA w czystym JavaScript, HTML i CSS.

## Wymagania (checklista)
- [ ] Minimum 3 widoki
- [ ] Routing między widokami
- [ ] PWA (manifest + service worker)
- [ ] Działanie offline
- [ ] Geolokalizacja
- [ ] Kamera / zdjęcia
- [ ] Responsywność
- [ ] Hosting HTTPS
- [ ] Strategia cache
- [ ] Lighthouse 80+
- [ ] Dokumentacja
